# NutriTrack — Архитектура проекта

> Мобильное PWA-приложение для ведения дневника питания с подсчётом БЖУ и калорий.
> Стек: Vanilla JS · HTML · CSS · localStorage · Chart.js

---

## Обзор

NutriTrack — полностью клиентское приложение без бэкенда. Все данные хранятся в `localStorage` браузера. Интерфейс оптимизирован под мобильные устройства (max-width: 480px) и поддерживает установку как PWA на iOS/Android.

## Три вкладки приложения

### 📋 Дневник (`#page-diary`)

Главный экран. Открывается по умолчанию.

- **Полоса дат** — переключение между днями (← →)
- **SVG-кольца** — 4 круговых индикатора прогресса: Белки / Жиры / Углеводы / Калории
- **Карточка итогов** — суммарные граммы и ккал за день
- **Список записей** — съеденные продукты с макросами и кнопкой удаления
- **Модальное окно добавления** — два режима:
  - Поиск по [[foods-db.js]] с выбором веса порции
  - Ручной ввод произвольного продукта

### 👤 Профиль (`#page-profile`)

- Персональные данные: имя, возраст, рост, вес, ИМТ (авторасчёт)
- **Три режима цели** с автоподбором норм БЖУ:
  - ✂️ Сушка — дефицит калорий, акцент на белок
  - ⚖️ Поддержание — балансированные нормы
  - 💪 Набор — профицит, высокие углеводы
- **Статистика:** стрик (серия дней), количество дней с записями, средние ккал за 7 дней
- **График Chart.js** — стопчатая диаграмма БЖУ по дням недели

### 🥗 Нутрициолог (`#page-nutri`)

- **Совет дня** — 14 советов, ротация по числу месяца
- **Персональный анализ** — 7 автоматических проверок рациона за 7 дней:
  - Уровень белка (vs. цель)
  - Превышение / дефицит калорий
  - Доля жиров (норма ≤ 35%)
  - Наличие овощей в рационе
  - Наличие рыбы в рационе
  - Разнообразие категорий (< 3 = предупреждение)
  - Избыток углеводов
- **Q&A** — ответы по ключевым словам (keyword-matching, без LLM)

## Хранилище данных

Всё хранится в `localStorage` без сервера и авторизации.

| Ключ | Тип | Содержимое |
|---|---|---|
| `bju_log_YYYY-MM-DD` | `Array<Entry>` | Записи питания за конкретный день |
| `bju_profile` | `Object` | Профиль пользователя и цели |

### Структура `Entry`
```js
{
  name: string,      // название продукта
  weight: number,    // вес порции в граммах
  proteins: number,  // граммы белка
  fats: number,      // граммы жиров
  carbs: number,     // граммы углеводов
  calories: number   // ккал
}
```

### Структура `Profile`
```js
{
  name: string, age: number, weight: number, height: number,
  goal: 'cut' | 'maintain' | 'bulk',
  targetP: number, targetF: number, targetC: number, targetK: number
}
```

## База продуктов (`foods-db.js`)

100 продуктов, сгруппированных в 12 категорий. Данные — на 100г продукта.

| Категория | `category` |
|---|---|

| Рыба и морепродукты | `fish` |
| Молочные продукты и яйца | `dairy` |
| Крупы и злаки | `grains` |
| Овощи | `vegetables` |
| Фрукты и ягоды | `fruits` |
| Орехи и семена | `nuts` |
| Масла | `fats` |
| Бобовые | `legumes` |
| Готовые блюда / Фастфуд | `fast` |
| Напитки | `drinks` |
| Сладкое | `sweets` |
